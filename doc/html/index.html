Introduction

Welcome to the OpenNSL API Guide and Reference Manual. This documentation is divided into sections accessed using the tabs at the top of the page. These sections are described below.

OpenNSL Documentation

Contains a list of documents including an overview of OpenNSL and information necessary for installing and configuring the software.

API Reference

Documents the OpenNSL APIs grouped by module name. The API Reference tab provides a hyperlinked list of all modules used to generate the documentation.

Files

Documents the OpenNSL APIs grouped by file name. The File List tab provides a hyperlinked list of all files used to generate the documentation. The Globals tab provides an index of all the OpenNSL API functions.
